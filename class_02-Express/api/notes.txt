Few steps to do before starting.

1. Initiate a new project.
npm init --y

2. Install typescript and ts-node as dev dependencies.
npm i -D typescript ts-node

3. Create tsconfig.json

4. Install types for node.
npm i -D @types/node

5. Install ExpressJS
npm i express

6. Install types for ExpressJS
npm i -D @types/express

To start this current project, all you really need is npm install, these are just notes for when you will want to try stuff on your own.
